IRRGARTEN

Lenke die Katze mit dem Finger

Scratch · Spiel 3
📱 Aufs iPad holen

Scanne den QR-Code mit der iPad-Kamera, dann hast du diese Aufgabe direkt vor dir. Tippe dann auf den orangen Knopf, um Scratch zu öffnen.

▶ Scratch öffnen

🎯 Darum geht es

Du baust einen Irrgarten und steuerst die Katze mit dem Finger hindurch: Sie läuft immer dorthin, wo du die Bühne berührst. Berührt die Katze eine schwarze Wand, wird sie gestoppt und ein Stück zurückgeschickt. Schaffst du es durch das ganze Labyrinth?

Block-Farben

  • Ereignisse
  • Steuerung
  • Bewegung
  • Fühlen
1

Katze kleiner machen

Größe
40

Öffne eine neue Scratch-Datei. Die Katze ist noch zu groß für den Irrgarten.

Klicke im Figuren-Bereich auf das Feld Größe, ändere die Zahl von 100 auf 40 und tippe danach auf eine freie Stelle neben dem Feld.

2

Zur Bühne wechseln

Jetzt brauchst du eine Spielfläche zum Malen. Klicke im Figuren-Bereich rechts auf den Button Bühne.

Klicke dann oben links auf den Reiter Bühnenbilder.

3

Zum Zeichnen umstellen

Klicke unten links auf In Rastergrafik umwandeln.

Wähle dann links in der Werkzeugleiste das Rechteck-Werkzeug aus.

Warum Rastergrafik? So kannst du frei mit Rechtecken malen, ohne dass sich die Formen später verschieben.

4

Den Irrgarten malen

Zeichne mit dem Rechteck-Werkzeug die Wände deines Irrgartens. Ziehe dafür einfach mehrere Rechtecke auf die Bühne.

Hast du dich vermalt? Nimm den Radiergummi oder klicke auf Rückgängig.

Tipp: Lass an den Öffnungen genug Platz, damit die Katze hindurchpasst. Das Spiel soll knifflig sein, aber nicht unmöglich.

5

Code für die Katze: dem Finger folgen

wenn ⚑ angeklickt
wiederhole fortlaufend
falls Maustaste gedrückt? , dann
drehe dich zu Mauszeiger ▾
gehe 4 er-Schritt

Klicke im Figuren-Bereich auf Figur1 (die Katze) und dann oben auf den Reiter Skripte. Baue dieses eine Skript. Es lässt die Katze immer dorthin laufen, wo du die Bühne berührst.

Du findest die Blöcke hier: wenn ⚑ angeklickt → Ereignisse, wiederhole fortlaufend und falls … dann → Steuerung, Maustaste gedrückt? → Fühlen, die beiden letzten → Bewegung.

So steuerst du: Auf dem iPad ist dein Finger der Mauszeiger: Halte den Finger auf die Bühne, und die Katze läuft dorthin. Am PC hältst du stattdessen die linke Maustaste gedrückt. Tipp: Bleib mit dem Finger immer ein Stück vor der Katze, dann läuft sie ruhig.

6

Zwischendurch testen

Klicke auf die grüne Flagge ⚑, um deinen bisherigen Code zu testen.

Die Katze sollte jetzt deinem Finger folgen, solange du die Bühne berührst. Aber Achtung: noch kann sie einfach durch die Wände gehen!

7

Die Katze an der Wand stoppen

wenn ⚑ angeklickt
wiederhole fortlaufend
falls wird Farbe berührt? , dann
drehe dich ↻ um 180 Grad
gehe 10 er-Schritt

Mit diesem Code prüft die Katze immer wieder, ob sie eine schwarze Wand berührt. Ist das der Fall, dreht sie sich um und geht ein Stück zurück.

Du findest die Blöcke hier: wenn ⚑ angeklickt → Ereignisse, wiederhole fortlaufend und falls … dann → Steuerung, wird Farbe … berührt? → Fühlen, die beiden letzten → Bewegung.

Wandfarbe einstellen: Tippe im Block wird Farbe … berührt? auf das kleine Farbfeld. Wähle mit der Pipette genau die Farbe deiner Wände (Schwarz). Nur so erkennt die Katze die Wand.

Fertig! Klicke auf die grüne Flagge ⚑ und teste dein Spiel. Viel Spaß im Irrgarten! 🎉
