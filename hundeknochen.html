HUNDEKNOCHEN

Sammel-Spiel mit Punktestand in Scratch

Scratch · Spiel
📱 Aufs iPad holen

Scanne den QR-Code mit der iPad-Kamera, dann hast du diese Aufgabe direkt vor dir. Tippe dann auf den orangen Knopf, um Scratch zu öffnen.

▶ Scratch öffnen

🎯 Darum geht es

Ein Hund läuft mit dem Finger über die Bühne und frisst Knochen. Eine Variable zählt die Punkte: Für jeden Knochen, den der Hund berührt, gibt es einen Punkt mehr. Durch wechselnde Kostüme sieht es aus, als würde der Hund laufen.

Block-Farben

  • Ereignisse
  • Steuerung
  • Bewegung
  • Aussehen
  • Fühlen
  • Variablen
1

Neue Datei — Katze löschen

Duplizieren
Exportieren
Löschen

Öffne eine neue Scratch-Datei. Die Katze brauchst du für dieses Spiel nicht.

Halte im Figuren-Bereich das Katzen-Symbol lange gedrückt (am PC: Rechtsklick) und tippe dann auf Löschen.

2

Einen Hund einfügen

🐶+

Jetzt setzt du den Hund als Spielfigur ein.

Tippe unten rechts auf das Symbol Figur wählen (sieht aus wie der Knopf links). Scrolle durch die Figuren und tippe auf Dog2.

3

Eine Variable "Punkte" anlegen

Punkte

Damit du den Punktestand speichern kannst, brauchst du eine Variable — das ist ein kleiner Behälter für eine Zahl.

Tippe auf die Gruppe Variablen → Neue Variable. Nenne sie Punkte und tippe auf OK.

4

Code für den Hund

wenn ⚑ angeklickt
setze Punkte ▾ auf 0
wiederhole fortlaufend
drehe dich zu Mauszeiger ▾
gehe 2 er-Schritt
wechsle zum nächsten Kostüm

Tippe zuerst den Hund an, damit dieser Code zu ihm gehört. Ziehe dann die Blöcke.

setze Punkte auf 0 → Gruppe Variablen, wechsle zum nächsten Kostüm → Gruppe Aussehen. Der Rest kommt aus Ereignisse, Steuerung und Bewegung.

So wirkt der Code: Beim Start wird Punkte auf 0 gesetzt. Danach dreht sich der Hund immer zum Finger, geht 2 Schritte und wechselt das Kostüm — dadurch bewegt er die Beine, als würde er laufen.

Finger-Tipp: Halte den Finger immer ein Stück vor dem Hund, dann läuft er dir schön hinterher.

5

Einen Knochen malen

🖌️+

Jetzt malst du den Knochen, den der Hund frisst.

Halte den Finger über Figur wählen und tippe auf Malen (Pinsel-Symbol). Tippe oben rechts auf In Rastergrafik umwandeln.

Male einen Knochen, der das Malfeld fast ganz ausfüllt (etwa 90 %), und fülle ihn mit Farbe.

6

Knochen kleiner machen

Größe 20

Der Knochen ist noch viel zu groß. Setze im Figuren-Bereich unter der Bühne die Größe der Knochen-Figur auf 20.

7

Code für den Knochen

wenn ⚑ angeklickt
zeige dich
wiederhole fortlaufend
falls wird Dog2 ▾ berührt? , dann
ändere Punkte ▾ um 1
verstecke dich

Tippe oben auf Skripte. Achtung: Dieser Code gehört zum Knochen, nicht zum Hund! Wähle vorher die Knochen-Figur aus.

zeige dich und verstecke dich → Aussehen, ändere Punkte um 1 → Variablen. Den Block wird … berührt? (Gruppe Fühlen) ziehst du in die sechseckige Öffnung von falls … dann.

So wirkt der Code: Der Knochen ist beim Start sichtbar. Sobald er Dog2 berührt, gibt es 1 Punkt mehr und der Knochen versteckt sich — als hätte der Hund ihn gefressen.

8

Mehr Knochen zaubern

Kopieren
Duplizieren
Exportieren
Löschen

Damit der Hund mehr fressen kann, brauchst du mehr Knochen. Du kopierst einfach den ersten — jeder neue Knochen hat dann schon den richtigen Code.

Halte den Knochen auf der Bühne lange gedrückt (am PC: Rechtsklick) → Duplizieren. So machst du insgesamt fünf Knochen und verteilst sie über die Bühne.

9

Testen

Tippe auf die grüne Flagge ⚑ und führe den Hund mit dem Finger zu den Knochen.

Prüfe genau: Erhöht sich der Punktestand jedes Mal um 1, wenn der Hund einen Knochen frisst? 🎉
