1

Variablen: der Punktestand

Computerprogramme arbeiten mit "Variablen", um Daten oder Informationen zu speichern. In Variablen speicherst du z.B. den Punktestand oder das Tempo, mit dem sich eine Figur bewegt. Anders als normale Zahlen können Variablen ihren Wert ändern: Wenn du einen Punkt machst, geht die Punkte-Variable um 1 hoch.

Merke: Eine Variable gleicht einem Kasten, der etwas Wichtiges enthält. Auf dem Kasten steht ein Name (z.B. "Punkte") — und was drin liegt, kann sich ändern.

So erstellst du die Variable:

1. Tippe links auf die Gruppe Variablen.   2. Tippe auf den Knopf Neue Variable.   3. Tippe Punkte ins Namensfeld.   4. Tippe auf OK.

2

Bei Spielbeginn: Punkte auf 0

wenn ⚑ angeklickt
setze Punkte ▾ auf 0

Lege den Block setze Punkte auf an den Start deines Programms. So beginnt der Zähler bei jedem Spielstart wieder bei 0.

Sonst würden die Punkte vom letzten Spiel einfach weiterzählen!

3

Punktzahl erhöhen

ändere Punkte ▾ um 1

Der Block ändere Punkte um 1 erhöht die Punkte-Variable jedes Mal um 1.

Wichtig: Dieser Code muss immer genau dann laufen, wenn du im Spiel einen Punkt gewinnst — z.B. wenn deine Figur etwas fängt.

4

Bei Spielende: Punkte anzeigen

sage Punkte

Scratch zeigt Variablen normalerweise oben links auf der Bühne an. Mit dem Block sage aus der Gruppe Aussehen kann die Figur die Punkte am Spielende zusätzlich in einer Sprechblase zeigen.

Achtung: Tippe nicht das Wort "Punkte" als Text ein! Ziehe den ovalen Punkte-Block aus der Gruppe Variablen in die Öffnung des sage-Blocks.

5

Noch schicker: Text + Punkte verbinden

sage verbinde Deine Punkte: und Punkte

So sagt die Figur z.B. "Deine Punkte: 4" — Text und Zahl zusammen:

1. Ziehe sage aus der Gruppe Aussehen.   2. Ziehe einen verbinde-Block aus den Operatoren in den sage-Block.   3. Tippe Deine Punkte: links in den verbinde-Block.   4. Ziehe den ovalen Punkte-Block in die rechte Öffnung.

6

Zufallszahlen wählen

gehe zu x: Zufallszahl von -200 bis 200 y: 0

Wenn Spiele immer gleich ablaufen, wäre das ziemlich langweilig! Bei einem Brettspiel entscheidet z.B. der Würfel, wie weit ein Spieler gehen darf — niemand kann vorhersehen, welche Zahl zwischen 1 und 6 kommt. Beim Computerspiel lassen wir den Computer genauso eine Zufallszahl wählen.

Im Block links: Die erste Zahl (-200) ist die niedrigste, die zweite Zahl (200) die höchste Zahl, die gewählt werden kann.

Probiere es aus: Baue den Code und tippe direkt auf den Block — die Figur springt jedes Mal an eine andere Stelle. Teste verschiedene Werte! Bei Fang den Donut lässt genau dieser Code die Donuts immer von einer neuen Stelle fallen.

7

Animationen: zwei Kostüme

Für Wolken jagen soll es aussehen, als drehe sich der Propeller des Doppeldeckers. Der Trick: Wir erstellen zwei "Kostüme" für das Flugzeug — eines mit Propeller oben und eines mit Propeller unten. Scratch wechselt dann schnell zwischen beiden hin und her. Mit dieser Technik kannst du eigene Animationen erstellen, z.B. tanzende Blumen oder winkende Hunde!

8

Kostüm duplizieren und abändern

duplizieren
exportieren
löschen

1. Male zuerst die Figur, die du animieren willst (z.B. den Doppeldecker).

2. Wechsle oben links auf den Reiter Kostüme. Halte Kostüm1 in der Kostüm-Liste lange gedrückt (am PC: Rechtsklick) und tippe auf Duplizieren. Die Figur hat nun zwei Kostüme.

3. Ändere das zweite Kostüm etwas ab: Tippe auf das Auswahl-Werkzeug (Pfeil) und ziehe einen Auswahlrahmen um den Propeller.

4. Tippe oben im Malbereich auf Vertikal spiegeln und ziehe den Propeller nach unten.

9

Kostüme wechseln = Animation!

wechsle zum nächsten Kostüm

Tippe mehrmals auf den Block wechsle zum nächsten Kostüm (Gruppe Aussehen) — Scratch springt zwischen den beiden Kostümen hin und her. Der Propeller "dreht" sich!

Tipp: In einem Spiel steckst du diesen Block in eine wiederhole fortlaufend-Schleife, dann läuft die Animation von ganz allein.
